[Color:Red]
使用方法：
[Color:White]
XScritp.exe [源文件] [/NoLogo] [/Debug]
    /NoLogo     不显示版权信息
    /Debug (/D) 调试模式
[Color:Magenta]
温馨提醒：XScript会自己添加到资源管理器右键菜单中的“发送到”中，便于使用！

[Color:Red]
脚本格式：
[Color:Green]
一、简易模式
[Color:White]
    直接书写脚本代码行，不得使用函数。
    如：
[Color:Magenta]
    Console.WriteLine("Hello NewLife!");
[Color:Green]
二、完整模式
[Color:White]
    代码必须写在方法之中，主函数必须是static void Main()
    如果没有写命名空间和类名，程序将自动添加
    如果需要引入命名空间，请务必书写完整的命名空间和类名，确保是完整的类文件
    如：
[Color:Magenta]
    static void Main() {
        Test();
    }

    static void Test() {
        Console.WriteLine("Hello NewLife!");
    }
